# TD买入策略 - 回测分析报告

**分析日期**: 2026-03-26  
**数据范围**: 2025年1月 - 2025年12月  
**初始资金**: 100,000元  
**仓位管理**: 每次买入10%仓位，最多同时持有10只股票

## 一、策略概述

### 信号定义
- **买入信号**: 趋势从非上升状态转入上升状态（up, up_natural, up_rally, up_secondary, up_break）
- **数据来源**: 全市场3552只股票分钟级趋势数据

### 回测数据说明
- 使用 `backtest_2025_strategy_A_TD.csv` 作为回测数据
- 持有1-10天的收益已预计算在回测文件中
- 2025年共有374笔TD买入信号

## 五、核心结论

### 5.1 止损策略效果不佳
- 止损会"卖飞"很多后续会反弹的票
- 止损触发率30-45%，大幅降低交易频率和总收益
- 亏损交易的平均亏损（-3.43% ~ -4.19%）与不止损时的平均亏损基本一致

### 5.2 最优策略：不止损 + 适度止盈
- **持有期限**: 8天
- **止盈设置**: +15% ~ +20%
- **止损设置**: 无
- **预期年收益**: +20%

### 5.3 原理分析
- TD买入信号在上升趋势初期入场
- 趋势延续概率较高（胜率56.7%）
- 即使短期回调，8天内大概率恢复
- 止盈设置在15-20%，可以让利润充分奔跑
- 不设止损，让亏损的票自然恢复

---

## 六、实盘建议

### 操作规则
1. 每天收盘后检查是否有TD买入信号
2. 信号出现次日开盘买入，最多持有10只
3. 每只股票仓位为账户的10%
4. 持有8天后自动卖出
5. 若期间涨幅达到15%，触发止盈卖出
6. 不设止损

### 注意事项
1. 这是2025年单年数据，样本有限
2. 历史表现不代表未来收益
3. 建议先小资金实盘验证
4. 考虑交易成本（佣金、印花税、滑点）
