# 利弗莫尔趋势规则（合并版）

---

## 一、趋势代号（10种）

### 上升趋势体系（5种）

| 代号 | 趋势名称 | 说明 |
|------|----------|------|
| up | 上升趋势 | 主上升趋势 |
| up_natural | 自然回撤 | 上升趋势中的正常回调 |
| up_rally | 回升 | 回调后的反弹 |
| up_secondary | 次级回撤 | 二次回调，幅度较小 |
| up_break | 破碎 | 趋势破坏，可能反转 |

### 下跌趋势体系（5种）

| 代号 | 趋势名称 | 说明 |
|------|----------|------|
| down | 下跌趋势 | 主下跌趋势 |
| down_natural | 自然回升 | 下跌趋势中的正常反弹 |
| down_rally | 回撤 | 反弹后的回调 |
| down_secondary | 次级回升 | 二次反弹，幅度较小 |
| down_break | 破碎 | 趋势破坏，可能反转 |

## 四、趋势转换规则

### 上升趋势体系

#### 1. up（上升趋势）
- **关键点**: key_high
- **转换规则**:
  - 价格 > key_high → 更新 key_high（继续 up）
  - 价格 < key_high × 0.94 → 转为 up_natural

#### 2. up_natural（自然回撤）
- **关键点**: n_low
- **转换规则**:
  - 价格 > n_low × 1.06 → 转为 up_rally
  - 价格 < n_low → 更新 n_low（继续 up_natural）

#### 3. up_rally（回升）
- **关键点**: n_low, key_high, rally_high
- **转换规则**:
  - 价格 < rally_high × 0.94 且 ≥ n_low → 转为 up_secondary
  - 价格 > rally_high 且 < key_high → 更新 rally_high
  - 价格 > key_high → 转为 up，更新 key_high
  - **特殊情况**：如果从n_low上涨超过6%（但不到6.4%），然后价格跌破n_low，可以直接转为up_secondary（不再要求从rally_high下跌6%）

#### 4. up_secondary（次级回撤）
- **关键点**: n_low, rally_high, secondary_low
- **转换规则**:
  - 价格 < secondary_low 且 > n_low → 更新 secondary_low
  - 价格 > secondary_low 且 < rally_high → 趋势不变
  - 价格 > secondary_low 且 > rally_high → 转为 up_rally，更新 rally_high
  - 价格 < secondary_low 且 < n_low → 转为 up_break，新增 break_low

#### 5. up_break（破碎）
- **关键点**: n_low, break_low
- **转换规则**:
  - 价格 < break_low 且 > n_low × 0.97 → 更新 break_low
  - 价格 < break_low 且 < n_low × 0.97 → 转为 down，新增 key_low
  - 价格 > break_low × 1.06 → 转为 up_rally，更新 rally_high，更新 n_low = break_low

---

### 下跌趋势体系

#### 6. down（下跌趋势）
- **关键点**: key_low
- **转换规则**:
  - 新低 → 更新 key_low
  - 价格 > key_low × 1.06 → 转为 down_natural

#### 7. down_natural（自然回升）
- **关键点**: n_high
- **转换规则**:
  - 新高 → 更新 n_high
  - 价格 < n_high × 0.94 → 转为 down_rally

#### 8. down_rally（回撤）
- **关键点**: n_high, key_low, rally_low
- **转换规则**:
  - 价格 > rally_low × 1.06 且 ≤ n_high → 转为 down_secondary
  - 价格 < rally_low 且 ≥ key_low → 更新 rally_low
  - 价格 < key_low → 转为 down
  - **特殊情况**：如果从n_high下跌超过6%（但不到6.4%），然后价格涨破n_high，可以直接转为down_rally（不再要求从rally_low回升6%）

#### 9. down_secondary（次级回升）
- **关键点**: n_high, rally_low, secondary_high
- **转换规则**:
  - 价格 > secondary_high 且 < n_high → 更新 secondary_high
  - 价格 > secondary_high 且 > n_high → 转为 down_break，新增 break_high
  - 价格 < secondary_high 且 > rally_low → 趋势不变
  - 价格 < secondary_high 且 < rally_low → 转为 down_rally

#### 10. down_break（破碎）
- **关键点**: n_high, break_high
- **转换规则**:
  - 价格 > break_high 且 < n_high × 1.03 → 更新 break_high
  - 价格 > break_high 且 > n_high × 1.03 → 转为 up
  - 价格 < break_high × 0.94 → 转为 down_rally
