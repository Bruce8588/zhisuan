## 同步流程

### 本地 → GitHub → 服务器

```
本地电脑  ──push──>  GitHub  ──pull──>  服务器
```

## 本地命令（Mac本地智算文件夹）

```bash
cd ~/openclaw/workspace/工作台/智算

# 1. 添加/修改文件后，暂存改动
git add .

# 2. 创建提交（写清楚改了什么）
git commit -m "修复了xxx问题"

# 3. 推送到GitHub
git push
```

## 服务器命令（阿里云服务器）

```bash
cd /root/智算

# 拉取最新代码
git pull

# 查看状态
git status
```

## 忽略的文件（不同步）

以下文件已加入 .gitignore，不会同步：
- `data/` - 数据库（太大）
- `output/` - 输出文件
- `venv311/` - Python环境
- `*.csv` - CSV数据文件
- `*.pdf` - PDF文档

## 首次设置（如果换电脑）

```bash
# 克隆仓库
git clone https://github.com/Bruce8588/zhisuan.git

# 进入目录
cd zhisuan
```

## 注意事项

1. 每次修改代码后，记得 `git add` + `git commit` + `git push`
2. 服务器上修改后，先 push 到 GitHub，本地再 pull
3. 如果冲突，先解决冲突再提交
