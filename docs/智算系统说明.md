# 智算系统资料汇总

## 智算1.0
- 路径: /root/2、智算/quant_trading_system/
- 功能: 量化交易系统，监控15只股票，检测交易信号
- 信号规则:
  - 买入: 趋势转为 up（上升趋势）
  - 卖出: 趋势转为 up_natural（自然回撤）
- 启动脚本: /root/2、智算/quant_trading_system/智算系统.sh
- Python环境: /root/2、智算/quant_env
