# 趋势体系详解

---

## 一、核心概念

### 1. 趋势系统是什么

趋势系统是一个**状态机**，用于根据股票价格变动来判断股票当前处于什么趋势状态。

```
状态机公式：
当前趋势 + 关键点 + 转换规则 = 下一个趋势
```

### 2. 三要素

| 要素 | 作用 |
|------|------|
| **当前趋势** | 股票现在处于什么状态 |
| **关键点** | 判断趋势转换的参考基准值 |
| **转换规则** | 当价格变动幅度达到一定条件时，趋势会切换到下一个状态 |

### 3. 关键点是什么

**关键点 = 判断趋势转换的"参考基准"**

每个趋势状态都有一个或几个关键点作为参考。当价格变动幅度超过这些关键点时，就会触发趋势的转换。

### 4. 6%规则

趋势系统使用**6%**作为判断幅度：

| 乘数 | 含义 | 用法 |
|------|------|------|
| × 0.94 | 下跌6% | 当价格跌破关键点的94%时触发 |
| × 1.06 | 上涨6% | 当价格涨破关键点的106%时触发 |

## 二、趋势体系总览

### 上升趋势体系（5种）

```
上升趋势 → 自然回撤 → 回升 → 次级回撤 → 破碎 → 可能反转 → 下跌趋势
```

| 趋势 | 中文名 | 说明 |
|------|--------|------|
| 上升趋势 | 上升趋势 | 主上升趋势，持续创新高 |
| 自然回撤 | 自然回撤 | 正常回调，幅度超过6% |
| 回升 | 回升 | 回调后的反弹，反弹幅度超过6% |
| 次级回撤 | 次级回撤 | 二次回调，幅度较小 |
| 破碎 | 破碎 | 趋势破坏，可能反转 |

### 下跌趋势体系（5种）

```
下跌趋势 → 自然回升 → 回撤 → 次级回升 → 破碎 → 可能反转 → 上升趋势
```

| 趋势 | 中文名 | 说明 |
|------|--------|------|
| 下跌趋势 | 下跌趋势 | 主下跌趋势，持续创新低 |
| 自然回升 | 自然回升 | 正常反弹，幅度超过6% |
| 回撤 | 回撤 | 反弹后的回调，回调幅度超过6% |
| 次级回升 | 次级回升 | 二次反弹，幅度较小 |
| 破碎 | 破碎 | 趋势破坏，可能反转 |

## 三、核心理解：股票涨跌 vs 趋势状态

### 重要前提

**股票具体只有两种状态：涨 or 跌**

**上升趋势体系和下跌趋势体系只是抽象意义的大方向，我们判断的大方向，但股票具体价格的变化是实际的涨跌。**

---

### 趋势 = 判断规则

趋势是一种**状态**，这个状态会决定：当价格实际涨跌时，如何判断下一个趋势。

---

### 价格变化时触发什么

**价格上涨时，会触发：**
| 当前趋势 | 动作 | 后续趋势 |
|---------|------|---------|
| 上升趋势 | 继续上升（创新高） | 上升趋势 |
| 自然回撤 | 转为回升 | 回升 |
| 次级回撤 | 转为回升 | 回升 |
| 破碎 | 转为回升 | 回升 |
| 下跌趋势 | 转为自然回升 | 自然回升 |
| 自然回升 | 更新高点 | 自然回升 |
| 次级回升 | 更新次级高点 | 次级回升 |
| 破碎 | 转为上升趋势 | 上升趋势 |

**价格下跌时，会触发：**
| 当前趋势 | 动作 | 后续趋势 |
|---------|------|---------|
| 上升趋势 | 转为自然回撤 | 自然回撤 |
| 自然回撤 | 更新回撤低点 | 自然回撤 |
| 回升 | 转为次级回撤 | 次级回撤 |
| 次级回撤 | 更新次级低点，或转为破碎 | 次级回撤/破碎 |
| 破碎 | 更新破碎低点，或转为下跌趋势 | 破碎/下跌趋势 |
| 下跌趋势 | 继续下跌（创新低） | 下跌趋势 |
| 自然回升 | 转为回撤 | 回撤 |
| 回撤 | 更新回撤低点 | 回撤 |
| 次级回升 | 转为回撤 | 回撤 |
| 破碎 | 转为回撤 | 回撤 |

## 五、每种趋势的详细说明

### 1. 上升趋势

**定义**：持续创新高的趋势

**所需关键点**：关键高点

**转换规则**：

| 条件 | 动作 | 后续趋势 |
|------|------|---------|
| 价格 > 关键高点 | 更新关键高点为新价格 | 继续上升趋势 |
| 价格 < 关键高点 × 0.94 | 转为自然回撤 | - |

---

### 2. 自然回撤

**定义**：上升趋势中的正常回调，幅度超过6%

**所需关键点**：自然回撤低点

**转换规则**：

| 条件 | 动作 | 后续趋势 |
|------|------|---------|
| 价格 > 自然回撤低点 × 1.06 | 转为回升 | - |
| 价格 < 自然回撤低点 | 更新自然回撤低点为新价格 | 继续自然回撤 |

---

### 3. 回升

**定义**：从回调低点反弹超过6%

**所需关键点**：自然回撤低点、关键高点、回升高点

**转换规则**：

| 条件 | 动作 | 后续趋势 |
|------|------|---------|
| 价格 < 回升高点 × 0.94 且 ≥ 自然回撤低点 | 转为次级回撤 | - |
| 价格 > 回升高点 且 < 关键高点 | 更新回升高点 | 继续回升 |
| 价格 > 关键高点 | 转为上升趋势，更新关键高点 | - |
| 价格从自然回撤低点上涨超过6%后跌破自然回撤低点 | 转为次级回撤（特殊情况） | - |

**特殊情况说明**：
如果从自然回撤低点反弹超过6%（但不到6.4%），然后又跌回自然回撤低点以下，说明这次反弹力度不够，直接进入次级回撤，不再要求从回升高点下跌6%。

---

### 4. 次级回撤

**定义**：二次回调，幅度较小

**所需关键点**：自然回撤低点、回升高点、次级回撤低点

**转换规则**：

| 条件 | 动作 | 后续趋势 |
|------|------|---------|
| 价格 < 次级回撤低点 且 > 自然回撤低点 | 更新次级回撤低点 | 继续次级回撤 |
| 价格 > 次级回撤低点 且 < 回升高点 | 趋势不变 | 继续次级回撤 |
| 价格 > 次级回撤低点 且 > 回升高点 | 转为回升 | - |
| 价格 < 次级回撤低点 且 < 自然回撤低点 | 转为破碎 | - |

---

### 5. 破碎

**定义**：趋势破坏，可能反转

**所需关键点**：自然回撤低点、破碎低点

**转换规则**：

| 条件 | 动作 | 后续趋势 |
|------|------|---------|
| 价格 < 破碎低点 且 > 自然回撤低点 × 0.97 | 更新破碎低点 | 继续破碎 |
| 价格 < 破碎低点 且 < 自然回撤低点 × 0.97 | 转为下跌趋势 | - |
| 价格 > 破碎低点 × 1.06 | 转为回升 | - |

---

### 6. 下跌趋势

**定义**：持续创新低的趋势

**所需关键点**：关键低点

**转换规则**：

| 条件 | 动作 | 后续趋势 |
|------|------|---------|
| 价格 < 关键低点 | 更新关键低点为新价格 | 继续下跌趋势 |
| 价格 > 关键低点 × 1.06 | 转为自然回升 | - |

---

### 7. 自然回升

**定义**：下跌趋势中的正常反弹，幅度超过6%

**所需关键点**：自然回升高点

**转换规则**：

| 条件 | 动作 | 后续趋势 |
|------|------|---------|
| 价格 > 自然回升高点 | 更新自然回升高点为新价格 | 继续自然回升 |
| 价格 < 自然回升高点 × 0.94 | 转为回撤 | - |

---

### 8. 回撤

**定义**：从反弹高点回落超过6%

**所需关键点**：自然回升高点、关键低点、回撤低点

**转换规则**：

| 条件 | 动作 | 后续趋势 |
|------|------|---------|
| 价格 > 回撤低点 × 1.06 且 ≤ 自然回升高点 | 转为次级回升 | - |
| 价格 < 回撤低点 且 ≥ 关键低点 | 更新回撤低点 | 继续回撤 |
| 价格 < 关键低点 | 转为下跌趋势 | - |

---

### 9. 次级回升

**定义**：二次反弹，幅度较小

**所需关键点**：自然回升高点、回撤低点、次级回升高点

**转换规则**：

| 条件 | 动作 | 后续趋势 |
|------|------|---------|
| 价格 > 次级回升高点 且 < 自然回升高点 | 更新次级回升高点 | 继续次级回升 |
| 价格 > 次级回升高点 且 > 自然回升高点 | 转为破碎 | - |
| 价格 < 次级回升高点 且 > 回撤低点 | 趋势不变 | 继续次级回升 |
| 价格 < 次级回升高点 且 < 回撤低点 | 转为回撤 | - |

---

### 10. 破碎

**定义**：趋势破坏，可能反转

**所需关键点**：自然回升高点、破碎高点

**转换规则**：

| 条件 | 动作 | 后续趋势 |
|------|------|---------|
| 价格 > 破碎高点 且 < 自然回升高点 × 1.03 | 更新破碎高点 | 继续破碎 |
| 价格 > 破碎高点 且 > 自然回升高点 × 1.03 | 转为上升趋势 | - |
| 价格 < 破碎高点 × 0.94 | 转为回撤 | - |
