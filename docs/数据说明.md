## 1. 日线原始数据

### data_daily/（2025年日线）
- **来源**: iFinD API via `fetch_data.py`
- **输入**: iFinD token
- **数据**: 2015-01-02 ~ 2026-03-24 日线OHLCV
- **说明**: 使用 iFinD HTTP API 获取，每日更新

### data_daily_2015_2024/（2015-2024年日线）
- **来源**: iFinD API via `fetch_daily_2015_2024.py`
- **输入**: iFinD token
- **数据**: 2015-01-05 ~ 2024-12-31 日线OHLCV
- **说明**: 历史数据存档，用于长期趋势分析

---

## 2. 指标数据（TD + MACD）

### data_daily_indicators/（2025年）
- **来源**: `calculate_indicators.py`
- **输入**: `data_daily/`
- **输出**: 每只股票的 TD九转序列 + MACD指标
- **指标说明**:
  - MACD: DIF = EMA12 - EMA26, DEA = DIF的9日EMA, MACD柱 = (DIF - DEA) * 2
  - TD九转: 连续9天收盘价低于/高于4天前收盘价

### data_daily_indicators_2015_2024/（2015-2024年）
- **来源**: `calculate_indicators_2015_2024.py`
- **输入**: `data_daily_2015_2024/`
- **输出**: 同上，2015-2024年数据
